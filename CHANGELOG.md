# Changelog

All notable changes to this project will be documented in this file.

## [v1.0.0] - 2026-03-07

### Added

- Initial release of the ALB Terraform module
- Application Load Balancer with configurable settings
- Target group support with health checks and stickiness
- HTTP and HTTPS listeners with TLS 1.3 support
- Listener rules with host header and path pattern conditions
- Additional certificate support via SNI
- Security group with configurable ingress rules
- S3 bucket for access logs with lifecycle policy
- WAF Web ACL association support
- Basic and complete usage examples
